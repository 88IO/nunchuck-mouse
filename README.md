# nunchuck-mouse
Pro Microを使ったWiiヌンチャクのUSBマウス実装。

詳細：https://scienceboy.jp/88io/2022/12/promicro-wii-nunchuck-mouse/

# 操作方法
少ないボタン数で基本機能を網羅するよう設計。また感覚的に操作できるように機能を割り当てた。

- Zボタン離し
  - ジョイスティック：カーソル移動
  - Cボタン：左クリック
- Zボタン押し
  - ジョイスティック：画面スクロール
  - Cボタン：右クリック

※ 中クリック未実装
